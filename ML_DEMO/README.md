# ML_DEMO Gesture Controller

This folder contains a standalone copy of the gesture controller demo.

## Requirements
- Python 3.11
- Windows (for `pycaw` and `screen-brightness-control`)
- A working webcam

## Install dependencies
```powershell
py -3.11 -m pip install -r requirements.txt
```

## Run
```powershell
py -3.11 Gesture_Controller.py
```

## Notes
- This demo includes only the gesture control module from the main repository.
- It does not include voice assistant or web UI code.
