# VisionDesk
VisionDesk is a gesture-controlled virtual mouse platform that uses Computer Vision and Artificial Intelligence to enable touchless computer interaction through real-time hand gesture recognition.

## Features

- Cursor Movement
- Left Click
- Right Click
- Double Click
- Drag and Drop
- Volume Control
- Brightness Control
- Vertical Scrolling
- Horizontal Scrolling
- Alt + Tab Application Switching
- Play / Pause Media
- Screenshot Capture
- FPS Counter
- Live Gesture HUD
- Help Overlay

## Technologies Used
- Python 3.11
- OpenCV
- MediaPipe
- PyAutoGUI
- PyCAW

## Future Enhancements

- Voice Commands: Enable hands-free control through speech recognition for executing mouse and system actions.

- Custom Gestures: Allow users to define and train their own gestures for personalized controls.

- Multi-Monitor Support: Extend cursor tracking and gesture functionality across multiple displays.

- AI Gesture Training: Use machine learning to learn user-specific hand movements and improve recognition accuracy over time.


## GESTURE CHEAT SHEET

- ✌ V Gesture → Move Cursor
- ☝ Index Finger → Right Click
- 🖕 Middle Finger → Left Click
- 🤏 Two Fingers Closed → Double Click
- ✊ Fist → Drag & Drop
- 🤏 Pinch (Major Hand) Horizontal → Brightness Control
- 🤏 Pinch (Major Hand) Vertical → Volume Control
- 🤏 Pinch (Minor Hand) Horizontal → Horizontal Scroll
- 🤏 Pinch (Minor Hand) Vertical → Vertical Scroll
- 🤙 Pinky Finger Only → Alt + Tab
- 💍 Ring Finger Only → Play / Pause Media
- 👍 Thumb Only → Screenshot Capture
- 🖐 Open Palm → Idle / Reset Cooldowns
- H Key → Toggle Help Overlay
- ENTER Key → Exit Program
